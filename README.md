# System_of_linear_equations


Данный проект реализует решение систем линейных алгебраических уравнений (СЛАУ) пятью различными методами: прямыми (Гаусс, LU-разложение, Холецкий) и итерационными (последовательная верхняя релаксация, сопряжённые градиенты). Решение функцией из библиотеки "Eigen" принимается за эталонное (необходимо подключить библиотеку). Матрица системы формируется на основе метода наилучшего приближения с учётом её ленточной структуры.

В программе реализуются следующие шаги:

1. Задание параметров:
- Концы отрезка [a, b]
- Эпсилон - точность итерационных методов 1e-8 
- Количество конечных элементов K
- Количество узлов N на конечном элементе
- Количество случайных внутренних точек L на конечном элементе
- Количество точек визуализации M_vis

2. Формирование СЛАУ

- Матрица и правая часть формируются один раз на основе метода наилучшего приближения
- Учитывается ленточная структура матрицы для оптимизации вычислений

3. Реализация пяти методов решения СЛАУ

- Гаусс: прямой метод исключения
- LU-разложение: разложение на нижнюю и верхнюю треугольные матрицы, обе записываются в копию исходной матрицы, единичная диагональ не хранится 
- Метод Холецкого: метод для симметричных положительно определённых матриц 
- Метод последовательной верхней релаксации: параметр релаксации омега оптимизирует сходимость 
- Метод сопряжённых градиентов: эффективен для разреженных СЛАУ 

4. Анализ результатов

- Вычисление абсолютной и относительной погрешности (относительно библиотечного решения)
- Вычисление отношения нормы невязки к норме правой части СЛАУ для всех методов
- Использование трех различных норм векторов (Манхэттенская (L_1), Евклидова (L_2), Чебышевская (L_inf))
